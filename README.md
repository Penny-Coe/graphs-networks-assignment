# Graphs, Networks & Systems Assignment

This repository contains the Colab notebook submission for the
Graphs, Networks & Systems module.

The project models the IMDB-5000 Movie Dataset as a similarity network to explore
network properties, centrality measures, community structure, and network-based
recommendation insights.

- Graphs_Networks_Systems_Assignment_report.ipynb
